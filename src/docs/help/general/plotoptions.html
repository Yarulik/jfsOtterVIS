Plotting Data

On the top left of the plotting area are buttons for some options.

  1. Printing the Plot
  2. Reset the Plot. With the left Mousebutton pressed down you can select an area of the plot for more details.
  3. Add Title etc
  4. Auto-ranging

For more details copy and visit https://ptolemy.eecs.berkeley.edu/java/ptplot5.10/ptolemy/plot/doc/index.htm

Select the X Range

Sometimes you want to display only a part of the x-range. To do this you select a rectangle by dragging the mouse. Now select the [use x-zoom] button. The selected x-range will now be used for displaying the plots.

If you select a new rectangle the selection will disapear.
